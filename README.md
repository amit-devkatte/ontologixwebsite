# Ontologix Software Website

Static responsive website for Ontologix Software Pvt Ltd.

## Files

- `index.html` - website markup and content
- `styles.css` - responsive design and visual styling
- `script.js` - mobile navigation and header behavior

## Publish

Upload these files to the web root for `www.ontologixsoftware.com`.

Update the email address in `index.html` if you want to use a different contact inbox.
